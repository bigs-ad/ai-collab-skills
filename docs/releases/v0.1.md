# v0.1 Candidate Release Notes

Status: Candidate for local installation and evaluation.

AI Collab Skills is a Codex skill suite for project collaboration, routing, planning, project management, delegation, bug and feature intake, and evidence-based review.

## Included

- Nine skills under `skills/`: `ai-collab`, `start-project`, `plan-work`, `manage-project`, `run-task`, `fix-bug`, `add-feature`, `delegate-work`, and `check-work`.
- Local installer at `scripts/install.sh`.
- English and Chinese README files.
- Baseline, forward-test, and Drift Guard evidence under `tests/`.
- Suite boundary review under `docs/suite-review.md`.

## Evidence-Supported Claims

This candidate supports these conservative claims:

- It provides a one-entry router through `ai-collab`.
- It standardizes project collaboration outputs around assumptions, evidence, blockers, and next action.
- It separates planning, delegation, execution, and acceptance decisions across child skills.
- It provides explicit evidence gates for handoff, release-readiness, bug fixes, new requirements, and acceptance reviews.
- It has passed local skill validation, installer syntax checks, install smoke testing, and multiple smoke baseline runs recorded in `tests/validation-results.md`.

## Not Claimed

Do not describe v0.1 as:

- mature
- bulletproof
- proven to reduce AI drift in production
- proven to outperform strong general assistants
- validated across many real repositories
- marketplace-ready

The current evidence shows useful structure and evidence-language consistency. It does not prove measurable superiority.

## Validation Summary

- All skill folders have passed `quick_validate.py`.
- `scripts/install.sh` has passed shell syntax checks and local clone/install smoke testing.
- The `v0.1.0` tag has passed temporary-target release install smoke testing.
- No-skill baseline testing did not produce clear RED failures.
- Drift Guard matched baselines preserved safety gates in both no-skill and with-skill arms.
- With-skill outputs were usually more explicit about assumptions, evidence, blockers, and next action.

## Release Notes Draft

Suggested GitHub release text:

```text
AI Collab Skills v0.1 is an early Codex skill suite for structured project collaboration.

It includes a one-entry router (`ai-collab`) plus child skills for project start, planning, project management, task execution, bug fixing, feature intake, delegation, and evidence-based review.

This release is suitable for local installation and evaluation under the MIT License. Baseline and smoke-test evidence is included, but the suite should still be treated as an early candidate rather than a mature or proven workflow system.
```

## Next Gates

Before a stronger release claim, run at least one of:

- Long-context recovery on a real project with state split across files and conversation history.
- One-turn unknown project takeover where source docs are sparse or partially stale.
- Repeated no-skill versus with-skill campaigns on real repositories.
- Real Codex App install, restart, and skill discovery verification.
