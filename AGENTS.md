# AGENTS.md

Use the user's language unless the user asks otherwise.

When editing this repository, use the Codex `skill-creator` guidance before changing skill folders.

Keep the suite general-purpose. Do not bake one project's domain rules into the public skills. Project-specific rules belong in that project's docs.

Keep `SKILL.md` files lean and put reusable templates under `assets/templates/`. Do not add README files inside individual skill folders.

After editing any skill, run `quick_validate.py` for that skill and inspect for placeholder text.
