# Example: Data Analysis Project

## Prompt

Use `$run-task`: "Analyze this CSV and tell me what changed this month."

## Expected Route

1. `run-task` classifies the work as data analysis.
2. It identifies source files, assumptions, and output format.
3. `check-work` verifies formulas, samples, and edge cases before conclusions are final.
4. `manage-project` is only needed if the analysis becomes a recurring project.

## Notes

Verification should focus on data provenance, transformation correctness, and whether conclusions are supported by evidence.
